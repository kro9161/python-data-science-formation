### 🟡 Option 2 – En local avec Anaconda (pour les utilisateurs avancés)

1. **Installez Anaconda**  
   Téléchargez-le depuis [https://www.anaconda.com/download](https://www.anaconda.com/download)  
   puis suivez les étapes d’installation (Python et Jupyter sont inclus).

2. **Récupérez le projet**
   - Ouvrez **Anaconda Navigator**
   - Lancez **Anaconda Prompt**
   - Tapez :
     ```bash
     git clone https://github.com/tonpseudo/python-data-science-formation.git
     ```
     (remplacez `tonpseudo` par votre pseudo GitHub si nécessaire)

3. **Ouvrez les notebooks**
   - Toujours depuis Anaconda Navigator :
     - Cliquez sur **Launch** en face de **Jupyter Notebook** ou **JupyterLab**  
     - Naviguez jusqu’au dossier `python-data-science-formation` que vous venez de cloner  
     - Ouvrez les notebooks (.ipynb) pour commencer à travailler

💡 *Astuce : vous pouvez exécuter toutes les cellules d’un notebook avec `Kernel > Restart & Run All`.*
